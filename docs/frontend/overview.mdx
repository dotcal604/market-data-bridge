# Dashboard (`/`)

The home page provides a snapshot of your trading system.

## Sections

### Stats Cards
Four KPI cards showing:
- **Total evaluations** — how many setups you've scored
- **Win rate** — from recorded outcomes
- **Avg R-multiple** — risk-adjusted return
- **Compliance** — % of trades that followed rules

### Edge Summary
Quick read on whether the ensemble is producing real edge — shows Sharpe ratio and edge score.

### Equity Curve
Cumulative R-multiple over time from recorded outcomes. Trending up = edge is real.

### Recent Evaluations
Last 10 evaluations with symbol, direction, ensemble score, and outcome status.

### Holly Alerts
Latest Holly AI alerts with strategy name, symbol, and auto-eval score (if enabled).
