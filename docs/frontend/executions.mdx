# Executions (`/executions`)

## Execution Stats
Summary bar showing:
- **Realized P&L** — net profit from today's closed trades
- **Commissions** — total fees paid
- **Fills** — number of execution fills
- **Shares** — total shares traded
- **Symbols** — unique tickers traded

## Execution Table
Each fill shows symbol, side, quantity, fill price, commission, and realized P&L. Filterable by symbol.
