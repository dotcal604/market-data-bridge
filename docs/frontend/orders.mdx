# Orders (`/orders`)

Real-time view of all orders on IBKR.

## Open Orders
Table showing order ID, symbol, action (BUY/SELL), order type, quantity, limit/stop price, status, and time-in-force. Updates via WebSocket.

## Completed Orders
Historical filled and cancelled orders with fill price, quantity, and completion time.
